🔧 이용 안내
모든 계산기는 무료인가요?
네, 모두의계산기의 모든 계산기는 완전 무료입니다. 회원가입 없이 바로 사용하실 수 있습니다.
모바일에서도 사용할 수 있나요?
네. 스마트폰, 태블릿, PC 모든 환경에서 사용 가능합니다. 별도 앱 설치 없이 브라우저에서 바로 이용하세요.
계산 결과를 공식 자료로 사용해도 되나요?
모두의계산기의 결과는 참고용입니다. 세금·보험료·퇴직금 등 중요한 금액은 국세청, 건강보험공단, 노무사 등 관련 기관에 꼭 확인하세요.
📌 자주 쓰는 계산기
자주 쓰는 계산기는 어떻게 저장되나요?
각 계산기 페이지의 ☆ 자주 쓰는 계산기 추가 버튼을 누르면 저장됩니다. 저장 위치는 브라우저의 로컬 스토리지(localStorage)로, 별도 서버에 전송되지 않습니다.
브라우저를 껐다 켜도 목록이 유지되나요?
네. 브라우저를 종료하거나 기기를 재시작해도 목록이 그대로 유지됩니다.
다른 기기에서도 목록이 보이나요?
아니요. 자주 쓰는 계산기 목록은 현재 기기의 브라우저에만 저장됩니다. 스마트폰과 PC 간 동기화는 지원하지 않습니다.
캐시를 삭제하면 목록도 사라지나요?
브라우저에서 사이트 데이터 삭제를 하면 목록이 사라질 수 있습니다. 단, 일반 캐시 삭제만으로는 보통 삭제되지 않습니다. 시크릿 모드에서는 탭을 닫으면 자동으로 삭제됩니다.
최대 몇 개까지 추가할 수 있나요?
홈 화면에는 최대 9개까지 표시됩니다. 9개를 초과해도 저장은 되지만 홈에서는 최근 9개만 보입니다.
📝 메모
메모는 어디에 저장되나요?
메모는 브라우저의 로컬 스토리지에 저장됩니다. 서버에 전송되지 않으며, 다른 페이지로 이동하거나 브라우저를 껐다 켜도 내용이 유지됩니다.
메모가 갑자기 사라졌어요
아래 경우에 메모가 삭제될 수 있습니다.

· 브라우저에서 사이트 데이터 삭제를 실행한 경우
· 시크릿 모드(프라이빗 브라우징)에서 사용한 경우
· 메모 패널 내 지우기 버튼을 누른 경우
📲 앱 설치 (홈 화면 추가)
설치하면 어떤 점이 좋은가요?
홈 화면에 아이콘이 생겨서 앱처럼 바로 실행할 수 있어요. 주요 장점은 아래와 같아요.

· 빠른 실행 — 브라우저 주소창 입력 없이 아이콘 한 번으로 바로 열려요
· 전체 화면 — 주소창·탭바 없이 앱처럼 넓게 표시돼요
· 오프라인 사용 — 인터넷 연결 없이도 주요 계산기를 사용할 수 있어요
· 빠른 로딩 — 자주 쓰는 파일이 기기에 캐시돼서 로딩이 훨씬 빨라요
· 완전 무료 — 앱스토어 없이 설치, 용량도 거의 안 차지해요
어떻게 설치하나요? (Android)
Android에서는 Chrome 브라우저로 접속해야 경고 없이 설치할 수 있어요.

방법 1 — 배너 이용
홈 화면 상단에 "홈 화면에 추가하면 앱처럼 바로 열려요!" 배너가 뜨면 [Chrome으로 설치] 버튼을 누르세요.

방법 2 — 수동 설치
Chrome 주소창 오른쪽 점 3개(⋮) → "앱 설치" 또는 "홈 화면에 추가" 선택

⚠️ 삼성 인터넷 등 다른 브라우저에서는 설치 시 경고가 뜰 수 있으니 Chrome을 사용해 주세요.
어떻게 설치하나요? (iPhone · iPad)
iOS에서는 Safari 브라우저로 접속 후 아래 방법을 따라주세요.

1. Safari 하단 가운데 공유 버튼(□↑) 탭
2. 스크롤해서 "홈 화면에 추가" 선택
3. 이름 확인 후 "추가" 탭

홈 화면에 모두의계산기 아이콘이 생성돼요.
어떻게 설치하나요? (PC · Mac)
PC에서는 Chrome 또는 Edge 브라우저 모두 지원해요.

Chrome
주소창 오른쪽 설치 아이콘(⊕) 클릭 → "설치" 클릭

Edge
주소창 오른쪽 설치 아이콘 클릭 → "설치" 클릭

설치 후 바탕화면·시작메뉴에 아이콘이 생기고, 별도 창으로 앱처럼 실행돼요.
오프라인에서도 정말 사용할 수 있나요?
네! 한 번 방문한 페이지는 기기에 저장되어 인터넷 없이도 계산이 가능해요.

단, 아래 기능은 인터넷이 필요해요.
· 환율 계산기 — 실시간 환율 데이터 조회
· 영문 주소 변환기 — 주소 API 호출
· 광고 — 오프라인 시 광고는 표시되지 않아요

그 외 연봉·세금·건강·날짜 등 대부분의 계산기는 오프라인에서도 정상 사용 가능해요.
설치한 앱을 삭제하려면 어떻게 하나요?
Android
홈 화면 아이콘 길게 누르기 → "제거" 선택

iPhone · iPad
홈 화면 아이콘 길게 누르기 → "앱 제거" → "홈 화면에서 제거"

PC (Chrome)
모두의계산기 앱 창 오른쪽 상단 점 3개(⋮) → "모두의계산기 제거"
또는 Chrome 주소창에 chrome://apps 입력 → 아이콘 우클릭 → "Chrome에서 제거"

PC (Edge)
모두의계산기 앱 창 오른쪽 상단 점 3개(⋮) → "제거"
