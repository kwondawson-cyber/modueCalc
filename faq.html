모두의계산기 이용 중 궁금한 점을 모았습니다

🔧 이용 안내
모든 계산기는 무료인가요?
네, 모두의계산기의 모든 계산기는 완전 무료입니다. 회원가입 없이 바로 사용하실 수 있습니다.
모바일에서도 사용할 수 있나요?
네. 스마트폰, 태블릿, PC 모든 환경에서 사용 가능합니다. 별도 앱 설치 없이 브라우저에서 바로 이용하세요.
계산 결과를 공식 자료로 사용해도 되나요?
모두의계산기의 결과는 참고용입니다. 세금·보험료·퇴직금 등 중요한 금액은 국세청, 건강보험공단, 노무사 등 관련 기관에 꼭 확인하세요.
📌 자주 쓰는 계산기
자주 쓰는 계산기는 어떻게 저장되나요?
각 계산기 페이지의 ☆ 자주 쓰는 계산기 추가 버튼을 누르면 저장됩니다. 저장 위치는 브라우저의 로컬 스토리지(localStorage)로, 별도 서버에 전송되지 않습니다.
브라우저를 껐다 켜도 목록이 유지되나요?
네. 브라우저를 종료하거나 기기를 재시작해도 목록이 그대로 유지됩니다.
다른 기기에서도 목록이 보이나요?
아니요. 자주 쓰는 계산기 목록은 현재 기기의 브라우저에만 저장됩니다. 스마트폰과 PC 간 동기화는 지원하지 않습니다.
캐시를 삭제하면 목록도 사라지나요?
브라우저에서 사이트 데이터 삭제를 하면 목록이 사라질 수 있습니다. 단, 일반 캐시 삭제만으로는 보통 삭제되지 않습니다. 시크릿 모드에서는 탭을 닫으면 자동으로 삭제됩니다.
최대 몇 개까지 추가할 수 있나요?
홈 화면에는 최대 9개까지 표시됩니다. 9개를 초과해도 저장은 되지만 홈에서는 최근 9개만 보입니다.
📝 메모
메모는 어디에 저장되나요?
메모는 브라우저의 로컬 스토리지에 저장됩니다. 서버에 전송되지 않으며, 다른 페이지로 이동하거나 브라우저를 껐다 켜도 내용이 유지됩니다.
메모가 갑자기 사라졌어요
아래 경우에 메모가 삭제될 수 있습니다.

· 브라우저에서 사이트 데이터 삭제를 실행한 경우
· 시크릿 모드(프라이빗 브라우징)에서 사용한 경우
· 메모 패널 내 지우기 버튼을 누른 경우
📲 앱 설치 (홈 화면 추가)
설치하면 어떤 점이 좋은가요?
홈 화면에 아이콘이 생겨서 앱처럼 바로 실행할 수 있어요. 주요 장점은 아래와 같아요.

· 빠른 실행 — 브라우저 주소창 입력 없이 아이콘 한 번으로 바로 열려요
· 전체 화면 — 주소창·탭바 없이 앱처럼 넓게 표시돼요
· 오프라인 사용 — 인터넷 연결 없이도 주요 계산기를 사용할 수 있어요
· 빠른 로딩 — 자주 쓰는 파일이 기기에 캐시돼서 로딩이 훨씬 빨라요
· 완전 무료 — 앱스토어 없이 설치, 용량도 거의 안 차지해요
어떻게 설치하나요? (Android)
Android에서는 Chrome 브라우저로 접속해야 경고 없이 설치할 수 있어요.

방법 1 — 배너 이용
홈 화면 상단에 "홈 화면에 추가하면 앱처럼 바로 열려요!" 배너가 뜨면 [Chrome으로 설치] 버튼을 누르세요.

방법 2 — 수동 설치
Chrome 주소창 오른쪽 점 3개(⋮) → "앱 설치" 또는 "홈 화면에 추가" 선택

⚠️ 삼성 인터넷 등 다른 브라우저에서는 설치 시 경고가 뜰 수 있으니 Chrome을 사용해 주세요.
어떻게 설치하나요? (iPhone · iPad)
iOS에서는 Safari 브라우저로 접속 후 아래 방법을 따라주세요.

1. Safari 하단 가운데 공유 버튼(□↑) 탭
2. 스크롤해서 "홈 화면에 추가" 선택
3. 이름 확인 후 "추가" 탭

홈 화면에 모두의계산기 아이콘이 생성돼요.
어떻게 설치하나요? (PC · Mac)
PC에서는 Chrome 또는 Edge 브라우저 모두 지원해요.

Chrome
주소창 오른쪽 설치 아이콘(⊕) 클릭 → "설치" 클릭

Edge
주소창 오른쪽 설치 아이콘 클릭 → "설치" 클릭

설치 후 바탕화면·시작메뉴에 아이콘이 생기고, 별도 창으로 앱처럼 실행돼요.
오프라인에서도 정말 사용할 수 있나요?
네! 한 번 방문한 페이지는 기기에 저장되어 인터넷 없이도 계산이 가능해요.

단, 아래 기능은 인터넷이 필요해요.
· 환율 계산기 — 실시간 환율 데이터 조회
· 영문 주소 변환기 — 주소 API 호출
· 광고 — 오프라인 시 광고는 표시되지 않아요

그 외 연봉·세금·건강·날짜 등 대부분의 계산기는 오프라인에서도 정상 사용 가능해요.
설치한 앱을 삭제하려면 어떻게 하나요?
Android
홈 화면 아이콘 길게 누르기 → "제거" 선택

iPhone · iPad
홈 화면 아이콘 길게 누르기 → "앱 제거" → "홈 화면에서 제거"

PC (Chrome)
모두의계산기 앱 창 오른쪽 상단 점 3개(⋮) → "모두의계산기 제거"
또는 Chrome 주소창에 chrome://apps 입력 → 아이콘 우클릭 → "Chrome에서 제거"

PC (Edge)
모두의계산기 앱 창 오른쪽 상단 점 3개(⋮) → "제거"

원하는 계산기가 없어요. 추가 요청할 수 있나요?

👷 노동 · 급여 계산
연봉 계산기 결과가 실제 급여명세서와 다른 이유가 뭔가요?
가장 흔한 원인은 비과세 항목입니다. 식대(월 20만원), 차량유지비, 육아수당 등 비과세 급여는 4대보험·소득세 산정 기준에서 제외됩니다. 비과세 항목이 클수록 실제 수령액이 계산기 결과보다 높습니다.

· 비과세 포함 연봉으로 입력했다면 → 계산 결과가 낮게 나옴
· 회사 내규상 추가 수당이 있다면 → 계산 결과가 낮게 나옴

계산기 내 비과세 항목 입력란에 식대 등 비과세 금액을 따로 입력하면 더 정확한 값을 얻을 수 있습니다.
실업급여 계산이 고용센터 안내와 다를 수 있나요?
네, 차이가 날 수 있습니다. 계산기는 입력한 평균임금 기준으로 산출하지만, 실제 지급액은 아래 요소에 따라 달라집니다.

· 상한액: 1일 68,100원
· 하한액: 최저임금의 80% (2026년 기준 66,048원/일)
· 이직확인서 처리 후 고용센터에서 최종 확정

특히 고임금 근로자는 상한액 적용으로 계산기 결과보다 낮게 지급될 수 있습니다.
종합소득세 계산기는 연말정산과 다른가요?
완전히 다른 개념입니다.

· 연말정산: 회사(원천징수의무자)가 대신 처리. 직장인 전용. 매년 1~2월.
· 종합소득세 신고: 프리랜서·사업자·투잡러(N잡러)가 직접 신고. 매년 5월 31일 마감.

종합소득세 계산기는 사업소득·기타소득이 있는 분들을 위한 도구입니다. 직장인 연말정산에는 연봉 계산기를 이용하세요.
주휴수당은 어떤 경우에 발생하나요?
주휴수당은 아래 조건을 모두 충족할 때 발생합니다.

· 주 15시간 이상 근무
· 소정근로일(약속한 근무일) 개근

주 15시간 미만 단시간 근로자이거나, 소정근로일에 결근이 있으면 주휴수당이 발생하지 않습니다. 주휴수당 계산기에 실제 근무시간을 입력하면 자동으로 반영됩니다.
퇴직금 계산기 결과가 실제 지급액과 다른 이유는?
퇴직금은 퇴직 전 3개월 평균임금을 기준으로 산정합니다. 평균임금 계산 시 연간 단위 지급 상여금·연차수당 일부도 포함되므로, 단순 월급 기준 계산보다 높게 나오는 경우가 많습니다.

계산기에 상여금·연차수당을 포함한 평균임금을 입력하면 더 정확한 결과를 얻을 수 있습니다. 최종 지급액은 회사 내규와 노무사 확인을 권장합니다.
🏦 대출 · 금융 계산
대출이자 계산기 결과가 은행 상담 내용과 다른 이유는?
계산기는 입력한 금리·기간의 단순 원리금 계산값입니다. 실제 대출에는 아래 요소가 추가로 반영됩니다.

· 우대금리: 급여이체, 자동이체, 신용카드 사용 조건 등
· 중도상환수수료: 조기 상환 시 추가 비용
· 변동금리: 금리 변동에 따른 이자 변동
· 취급수수료·보증료

계산기는 참고용으로만 사용하고, 최종 조건은 금융기관에 확인하세요.
전세 대출 계산기에서 DSR이 반영되나요?
현재 전세대출 계산기는 순수 이자·원리금 계산에 집중하며, 금융기관별 DSR(총부채원리금상환비율) 심사 기준은 반영하지 않습니다. DSR 기준은 은행별·상품별로 다르므로 실제 한도는 금융기관 상담이 필요합니다.
복리 계산기에서 세금은 반영되나요?
기본 계산은 세전 기준입니다. 실제 적금·예금 만기 수령액은 이자소득세 15.4%(소득세 14% + 지방소득세 1.4%)가 원천징수됩니다. 세후 수익을 확인하려면 계산 결과에서 이자 부분의 15.4%를 차감하시면 됩니다.
📋 계산 기준 및 정확도
2026년 적용 기준 수치는 어떻게 되나요?
2026년 모두의계산기 적용 기준값입니다.

· 최저임금: 10,320원/시
· 국민연금: 9.5% (근로자 4.75%, 사용자 4.75%)
· 건강보험: 7.19% (근로자 3.595%, 사용자 3.595%)
· 장기요양보험: 건강보험료의 13.14%
· 고용보험: 1.8% (근로자 0.9%, 사용자 0.9%)
· 실업급여 상한: 68,100원/일, 하한 66,048원/일
법령이 바뀌면 계산기가 언제 업데이트되나요?
최저임금·4대보험 요율 등 매년 바뀌는 수치는 연초(1월)에 일괄 업데이트합니다. 세율·공제 기준 등 연중 변경 사항은 관보 공시 후 최대한 빠르게 반영하며, 변경 내역은 홈 화면 업데이트 로그에서 확인할 수 있습니다.
계산 결과의 정확성은 어떻게 검증되나요?
모두의계산기는 아래 방법으로 계산식을 검증합니다.

· 공공기관 교차 검증: 국세청 홈택스 모의 계산, 4대보험 공단 계산기와 결과값 대조
· 법령 원문 확인: 국세청 고시, 고용노동부 고시, 관보 기준으로 계산식 설계
· 사용자 오류 제보 반영: 현업 종사자 제보 시 영업일 3일 이내 검토·수정

그럼에도 개인별 특수 상황(비과세 항목, 감면 특례 등)에 따라 차이가 날 수 있으므로 중요한 의사결정 시 전문가 확인을 권장합니다.

💬 원하는 답을 못 찾으셨나요?

계산기 추가 요청, 오류 제보, 계산식 개선 의견은 이메일로 보내주세요.
영업일 기준 3일 이내 답변드립니다.
